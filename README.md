# sed-ytd
YouTube Video Downloader Using  Sed  In A Five Line Shell Script 

# Requirements

* /bin/sh
* curl
* sed

# Installation
* git clone this repo
* chmod +x sed-ytd.sh


# Usage
```sh
./sed-ytd.sh 'https://www.youtube.com/watch?v=7R8OQOjqN1A'
```
